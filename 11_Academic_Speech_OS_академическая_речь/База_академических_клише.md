# База академических клише

## Назначение

Готовые устойчивые формулировки для введения, перехода, примера и вывода.

## Структура

1. Введение
2. Определение
3. Переход
4. Пример
5. Вывод
6. Ответ на уточнение

## Поля для заполнения

### Введение
- [НУЖНО ЗАПОЛНИТЬ]: «В данном вопросе речь идёт о...»

### Определение
- [НУЖНО ЗАПОЛНИТЬ]: «Под ... принято понимать...»

### Переход
- [НУЖНО ЗАПОЛНИТЬ]: «Далее следует рассмотреть...»

### Пример
- [НУЖНО ЗАПОЛНИТЬ]: «В качестве примера можно привести...»

### Вывод
- [НУЖНО ЗАПОЛНИТЬ]: «Таким образом, можно заключить, что...»

### Ответ на уточнение
- [НУЖНО ЗАПОЛНИТЬ]: «Если уточнить, то речь идёт о...»

## Связь с другими разделами

Используется в `Шаблон_устного_ответа.md` и архитектуре идеального ответа (`06`).
