# Академические клише для экзамена

## Назначение

Готовые речевые формулы для устного ответа комиссии.

## Структура

1. Введение
2. Определение
3. Переход
4. Пример
5. Методическое решение
6. Вывод

## Поля для заполнения

- **Введение:** «В данном вопросе речь идёт о...»
- **Определение:** «Под ... принято понимать...»
- **Переход:** «Далее следует рассмотреть...»
- **Пример:** «В школьной практике это проявляется в...»
- **Методика:** «Методически целесообразно...»
- **Вывод:** «Таким образом, можно заключить, что...»

## Связь с другими разделами

Связан с Academic Speech OS и устными формулами ответа.
