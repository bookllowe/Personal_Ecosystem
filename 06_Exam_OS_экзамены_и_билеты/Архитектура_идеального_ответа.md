# Архитектура идеального ответа для комиссии

## 1. Полная версия ответа

Используется для подготовки и письменной проработки билета.  
Должна быть развёрнутой, академической, доказательной.  
Объём: достаточно большой, чтобы раскрыть тему полностью.

## 2. Средняя версия ответа

Используется для устного ответа.  
Длительность: 5–7 минут.  
Содержит только главное:

- тезис;
- определения;
- 2–3 аргумента;
- пример;
- вывод.

## 3. Краткая версия ответа

Используется для быстрого повторения.  
Длительность: 1–2 минуты.  
Содержит:

- главное определение;
- центральную мысль;
- один пример;
- итоговый вывод.

## 4. Обязательные элементы ответа

- мотивационная связь с педагогическим образованием;
- терминология;
- современная научная или методическая литература;
- логика тезис — аргументы — пример — вывод;
- интеграция истории и педагогики;
- пример из школьной практики;
- собственная профессиональная позиция.

## 5. Ответ через критерии комиссии

Для каждого билета нужно отдельно проверить:

- как ответ показывает мотивационно-ценностный критерий;
- как ответ показывает когнитивный критерий;
- как ответ показывает операционно-деятельностный критерий.
