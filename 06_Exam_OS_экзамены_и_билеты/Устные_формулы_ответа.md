# Устные формулы ответа

## Назначение

Короткие формулы построения устного ответа.

## Структура

1. Формула 5–7 минут
2. Формула 1–2 минуты
3. Формула ответа на уточнение

## Поля для заполнения

- **5–7 минут:** Тезис → понятия → научная основа → аргументы → школьный пример → методическое решение → вывод
- **1–2 минуты:** Определение → центральная мысль → один пример → вывод
- **Уточнение:** Уточнение понятия → краткий аргумент → связь с практикой
