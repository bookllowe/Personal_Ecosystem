# 16_Портфолио_для_комиссии

## Назначение раздела

Портфолио для комиссии: документы, проекты, фрагменты уроков, связь с критериями.

## Что хранится в разделе

- Карта портфолио
- Сертификаты
- Проекты
- Фрагменты уроков
- Анализ тем ЕГЭ
- Связь с критериями
- Чеклист

## Основные файлы

- `README.md`
- `Карта_портфолио.md`
- `Сертификаты_и_грамоты.md`
- `Проекты_и_наработки.md`
- `Фрагменты_уроков.md`
- `Анализ_трудных_тем_ЕГЭ.md`
- `Связь_портфолио_с_критериями.md`
- `Что_показать_комиссии.md`
- `Чеклист_портфолио.md`

## Связь с другими разделами

Портфолио усиливает ответ, но учитывается при минимум 20 баллах за ответы.

## Как экономить токены

- Хранить длинные данные в markdown-файлах.
- Не отправлять в ИИ всю книгу целиком.
- Анализировать документы по главам.
- Использовать краткие карточки.
- Повторно использовать шаблоны.
- Хранить промежуточные результаты.
