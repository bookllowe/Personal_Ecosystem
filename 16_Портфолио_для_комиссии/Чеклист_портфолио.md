# Чеклист портфолио

## Назначение

Проверка готовности портфолио к комиссии.

## Структура

- [ ] Есть карта портфолио
- [ ] Есть связь каждого элемента с критерием
- [ ] Есть 1 сильный сертификат/документ
- [ ] Есть 1 проект/наработка
- [ ] Есть 1 фрагмент урока
- [ ] Есть 1 анализ трудной темы ЕГЭ
- [ ] Есть сценарий показа
- [ ] Понятно, что не показывать

## Поля для заполнения

- **Готовность портфолио:** низкая / средняя / высокая
- **Что дособрать:** [НУЖНО ЗАПОЛНИТЬ]

## Связь с другими разделами

Портфолио усиливает ответ, но не заменяет его (нужно минимум 20 баллов за ответы).
