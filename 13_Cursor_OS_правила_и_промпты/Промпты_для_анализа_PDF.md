# Промпты для анализа PDF

## Назначение

Готовые формулировки для разбора PDF по частям, без загрузки всей книги.

## Структура

1. Цель анализа
2. Промпт
3. Что сохранить
4. Связь

## Поля для заполнения

### Промпт 1. Обзор документа
«Сделай краткий обзор PDF: тема, структура, для чего полезен. Не пересказывай всё.»

### Промпт 2. Разбор главы
«Разбери только указанную главу/страницы по шаблону `Карточка_главы.md`.»

### Промпт 3. Извлечение знаний
«Извлеки 5–10 ключевых идей и сохрани по `Шаблон_извлечения_знаний.md`.»

**Документ:** [НУЖНО ЗАПОЛНИТЬ]  
**Фрагмент:** [страницы / глава]  
**Куда сохранить результат:** [НУЖНО ЗАПОЛНИТЬ]

## Связь с другими разделами

Результаты уходят в Research OS (`07`) и Knowledge OS (`08`).
