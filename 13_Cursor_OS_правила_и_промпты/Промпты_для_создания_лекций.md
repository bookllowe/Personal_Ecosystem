# Промпты для создания лекций

## Назначение

Быстро запускать создание полных академических лекций, а не кратких конспектов.

## Главное правило

Лекция должна быть рассчитана минимум на 15 минут чтения, быть академически сильной, доказательной и пригодной для подготовки к комиссии.

Для полного промпта используйте файл:

`Промпт_для_создания_полной_лекции.md`

## Краткий запуск

«Создай полную академическую лекцию по теме ... строго по `Стандарт_академической_лекции.md` и `Шаблон_полной_академической_лекции.md`. Нельзя делать краткий конспект вместо лекции. Синтезируй несколько источников. Добавь вопросы комиссии и три версии ответа: полную, среднюю устную и краткую.»
