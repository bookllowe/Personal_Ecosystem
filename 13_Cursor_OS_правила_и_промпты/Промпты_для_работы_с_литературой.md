# Промпты для работы с литературой

## Назначение

Промпты для анализа книг и извлечения знаний по частям.

## Структура

1. Структурная разведка
2. Разбор главы
3. Матрица извлечения
4. Синтез для лекции

## Поля для заполнения

### Промпт
«Проведи структурную разведку книги/главы ... Извлеки определения, тезисы, классификации и спорные точки. Не пересказывай всё. Заполни матрицу извлечения.»

- **Источник:** [НУЖНО ЗАПОЛНИТЬ]
- **Куда сохранить:** `07` / `08`
