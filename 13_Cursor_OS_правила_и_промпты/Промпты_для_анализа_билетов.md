# Промпты для анализа билетов

## Назначение

Промпты для разбора билета и подготовки трёх версий ответа.

## Структура

1. Разбор формулировки
2. Построение структуры
3. Три версии ответа
4. Вопросы комиссии
5. Проверка по критериям

## Поля для заполнения

### Промпт
«Разбери билет ... Создай карточку билета. Подготовь полную, среднюю и краткую версии. Добавь вопросы комиссии и проверь ответ через три критерия.»

- **Куда сохранить:** `06`
