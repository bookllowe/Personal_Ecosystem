# Правила экономии токенов

## Назначение

Снижать расход токенов и сохранять качество работы.

## Структура

1. Принципы
2. Что делать
3. Чего не делать
4. Рабочий порядок

## Поля для заполнения / правила

- Хранить длинные данные в markdown-файлах.
- Не отправлять в ИИ всю книгу целиком.
- Анализировать документы по главам.
- Использовать краткие карточки.
- Повторно использовать шаблоны.
- Хранить промежуточные результаты.
- В чат давать только нужный фрагмент + ссылку на файл.
- Сначала просить структуру, потом содержание.
- Полные лекции и развёрнутые ответы хранить в файлах; в чат не дублировать целиком без необходимости.

**Текущий режим экономии:** обычный / строгий  
**Заметка по последней задаче:** [НУЖНО ЗАПОЛНИТЬ]

## Связь с другими разделами

Базовое правило для всей системы. Детали промптов — в файлах этой папки.
