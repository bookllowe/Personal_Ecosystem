# Промпт для создания полной академической лекции

Используй этот промпт, когда нужно создать лекцию по теме или билету.

## Промпт

Создай полную академическую лекцию по теме: [ВСТАВИТЬ ТЕМУ].

Контекст:
Направление подготовки: Педагогическое образование.
Направленность: Историческое образование.
Цель: подготовка к вступительному экзамену / собеседованию / ответу комиссии.

Требования:
1. Лекция должна быть рассчитана минимум на 15 минут чтения.
2. Лекция должна быть академически сильной, содержательной, доказательной.
3. Нельзя делать краткий конспект вместо лекции.
4. Нельзя пересказывать одну книгу.
5. Нужно синтезировать материал из нескольких источников.
6. Нужно показывать понятия, подходы, классификации, дискуссии, примеры и выводы.
7. Нужно учитывать критерии комиссии:
   - мотивационно-ценностный;
   - когнитивный;
   - операционно-деятельностный.
8. Нужно показать связь темы с преподаванием истории.
9. Нужно включить примеры из школьной практики.
10. Нужно добавить возможные вопросы комиссии.

Структура лекции:
1. Паспорт лекции.
2. Место темы в курсе.
3. Цель.
4. Задачи.
5. Планируемые результаты.
6. Центральная проблема.
7. Ключевые понятия.
8. План лекции.
9. Введение.
10. Основная часть.
11. Заключение.
12. Опорный конспект.
13. Схемы и таблицы.
14. Вопросы для самопроверки.
15. Возможные вопросы комиссии.
16. Практические задания.
17. Литература.
18. Контроль качества.

Особое требование:
После лекции создай:
- краткую версию для повторения;
- среднюю устную версию на 5–7 минут;
- полный академический ответ;
- список терминов;
- список исследователей;
- список примеров из преподавания истории;
- академические клише для устного ответа.

## Куда сохранять результат

- Полная лекция — в рабочую папку темы / курса.
- Три версии ответа — в карточку билета (`06`).
- Источники и пробелы — в Research OS (`07`).
- Термины — в Словарь архитектора (`02`).
