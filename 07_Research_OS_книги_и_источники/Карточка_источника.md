# Карточка источника

## Назначение

Краткая карточка любого источника: статья, лекция, сайт, документ, видео.

## Структура

1. Тип и реквизиты
2. Суть
3. Надёжность
4. Применение
5. Связи

## Поля для заполнения

- **Тип источника:** книга / статья / лекция / документ / другое
- **Название:** [НУЖНО ЗАПОЛНИТЬ]
- **Автор / источник:** [НУЖНО ЗАПОЛНИТЬ]
- **Где лежит / ссылка:** [НУЖНО ЗАПОЛНИТЬ]
- **Кратко о чём:** [НУЖНО ЗАПОЛНИТЬ]
- **Насколько надёжен:** высокий / средний / низкий
- **Для каких тем / билетов / проектов:** [НУЖНО ЗАПОЛНИТЬ]
- **Что именно использовать:** [НУЖНО ЗАПОЛНИТЬ]

## Связь с другими разделами

Все источники должны быть привязаны к темам, билетам или проектам.
