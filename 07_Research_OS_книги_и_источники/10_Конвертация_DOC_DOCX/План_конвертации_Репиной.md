## Проблема

Файл находится в формате DOC. Cursor не извлекает из него текст напрямую. Для дальнейшей работы нужно конвертировать его в DOCX или Markdown.

## Цель конвертации

Получить читаемый файл для дальнейшего извлечения:
- оглавления;
- карты глав;
- тезисов;
- связи с билетами;
- цитат и страниц.

## Возможные способы

### Способ 1. Microsoft Word

1. Открыть DOC в Microsoft Word.
2. Сохранить как DOCX.
3. При необходимости сохранить копию как PDF.
4. Затем использовать DOCX/PDF для извлечения текста.

### Способ 2. LibreOffice

1. Открыть DOC в LibreOffice Writer.
2. Сохранить как DOCX.
3. При необходимости экспортировать в PDF.
4. Проверить качество текста.

### Способ 3. Pandoc

Использовать только если установлен Pandoc и он корректно читает DOC/DOCX.

## Рекомендуемый маршрут

DOC → DOCX → Markdown.
