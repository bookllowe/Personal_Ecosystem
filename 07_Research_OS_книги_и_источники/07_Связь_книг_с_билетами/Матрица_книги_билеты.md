# Матрица связи книг с билетами

## Назначение

Файл показывает, какие книги подходят к каким билетам.

## Правило

Одна книга может использоваться для многих билетов. Один билет должен опираться на несколько источников.

## Источник оригиналов

```text
C:\Users\User\Documents\Книги для магистратуры
```
