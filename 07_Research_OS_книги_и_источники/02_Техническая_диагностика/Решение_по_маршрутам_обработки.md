# Решение по маршрутам обработки книг

## Назначение

Файл фиксирует, каким способом каждая книга будет обрабатываться дальше.

## Типовые маршруты

| Маршрут | Когда применяется | Процесс |
|---|---|---|
| А. Текстовый PDF | текст извлекается нормально | PDF → Markdown → главы → тезисы |
| B. Скан без текста | текст не выделяется | PDF → OCR → Markdown → главы → тезисы |
| C. Плохой OCR | текст есть, но грязный | PDF → повторный OCR → очистка → Markdown |
| D. DOC/DOCX | файл Word | DOC → DOCX → Markdown |
| E. Сложные схемы/таблицы | много таблиц и схем | выборочные главы + ручная проверка таблиц |
| F. Только справочное использование | книга нужна редко | не конвертировать целиком, использовать отдельные фрагменты |

## Таблица маршрутов

| Книга | Диагноз | Маршрут | Почему | Следующее действие |
|---|---|---|---|---|
| Короткова / Студеникин | текстовый PDF со схемами и таблицами; OCR не нужен | E | много схем/таблиц; на страницах-схемах мало сплошного текста | извлечь оглавление; конвертировать только нужные главы; вручную проверить таблицы |
| Селиванова | текстовый PDF; качество хорошее | A | текст извлекается нормально | извлечь оглавление; выбрать главы по воспитанию |
| Смоленский | текстовый PDF; качество хорошее | A | плотный читаемый текст | извлечь оглавление; Markdown по главам |
| Тош | текстовый PDF; качество хорошее | A | текст извлекается нормально | извлечь оглавление; главы о факте/источнике/методе |
| Репина. История исторического знания | старый DOC был конвертирован в DOCX | D. DOCX → Markdown → оглавление → карта глав → связь с билетами | теперь файл доступен в более современном формате, который можно диагностировать и конвертировать | извлечь оглавление из DOCX |
| Данилевский | текстовый PDF; большой объём (398 стр.) | A | текст есть; OCR не нужен; нельзя грузить целиком | извлечь смысловое оглавление; Markdown только глав для 31–33 |

## Сводка

- Маршрут A: 4 книги (Селиванова, Смоленский, Тош, Данилевский).
- Маршрут D: 1 книга (Репина).
- Маршрут E: 1 книга (Короткова/Студеникин).
- Маршруты B и C: пока не назначены.
