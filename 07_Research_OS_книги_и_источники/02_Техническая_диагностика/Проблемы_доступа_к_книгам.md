# Проблемы доступа к книгам

## Назначение

Файл фиксирует проблемы, из-за которых Cursor или инструменты не могут прочитать книгу.

## Статус доступа на 2026-07-22

| Объект | Статус |
|---|---|
| Внешняя папка `C:\Users\User\Documents\Книги для магистратуры` | доступна |
| 6 файлов из реестра | найдены |
| PDF-текст через PyMuPDF | доступен для 5 PDF |
| DOC (Репина) | файл есть; прямой текст не извлечён |
| LibreOffice / Word / Pandoc в PATH | не обнаружены |

## Возможные проблемы

| Проблема | Как проявляется | Как исправить |
|---|---|---|
| Внешняя папка недоступна | Cursor не видит путь | открыть папку в workspace или скопировать файлы |
| PDF без текстового слоя | текст не выделяется | OCRmyPDF + Tesseract |
| Плохой OCR | текст с ошибками | повторный OCR, очистка текста |
| Старый DOC | файл не читается | DOC → DOCX через Word/LibreOffice |
| Сложные таблицы | Markdown ломает структуру | ручная проверка, отдельные таблицы |
| Две колонки | текст смешивается | Docling/Marker или ручная разметка |
| Сноски теряются | ссылки исчезают | сохранять цитаты отдельно |
| Большой файл | Cursor зависает или пропускает части | работать по главам |
| Дубликаты под разными именами | один файл учтён дважды | сверять хеш/размер; оставлять одно имя в реестре |
| Нет конвертеров DOC | нельзя сделать DOC → DOCX | установить LibreOffice/Word/Pandoc или конвертировать вручную |

## Зафиксированные проблемы по текущей шестёрке

| Книга | Проблема | Решение |
|---|---|---|
| Короткова/Студеникин | дубликат имени с файлом «1978»; много схем | в реестре указать одно каноническое имя; маршрут E |

| Данилевский | большой объём; закладки технические | работать по главам; оглавление извлекать отдельно |
