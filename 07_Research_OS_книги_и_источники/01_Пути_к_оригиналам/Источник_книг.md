# Источник книг

## Основная папка с книгами

```text
C:\Users\User\Documents\Книги для магистратуры
```

## Рабочий проект

```text
C:\Users\User\Documents\Личная_цифровая_экосистема
```

## Правило хранения

Оригинальные PDF, DOC и DOCX хранятся во внешней папке с книгами.

В проекте хранятся:
- реестр книг;
- техническая диагностика;
- оглавления;
- карты глав;
- Markdown-версии выбранных глав;
- тезисы;
- цитаты;
- связь с билетами;
- результаты анализа.

## Почему не копируем всё сразу

- Чтобы не перегружать проект.
- Чтобы не дублировать большие файлы.
- Чтобы сохранять оригиналы отдельно.
- Чтобы Cursor работал с аналитическими материалами, а не с хаотичной папкой PDF.
- Чтобы экономить токены.

## Когда можно копировать книгу в проект

Книгу можно скопировать в проект только если:
- она является базовой для многих билетов;
- её нужно конвертировать по главам;
- требуется OCR;
- нужно сохранить обработанную версию рядом с тезисами.

Если книга копируется, её помещать в:
`07_Research_OS_книги_и_источники/09_Проблемные_файлы_OCR`

или в отдельную техническую папку по решению пользователя.

## Конвейер

внешняя папка книг → реестр → диагностика → оглавление → карта глав → связь с билетами → выбор нужных глав → Markdown → тезисы → цитаты → билет.
