# Шаблон извлечения знаний

## Назначение

Вытащить из фрагмента текста только нужное знание в краткой форме.

## Структура

1. Откуда извлечено
2. Факт / идея
3. Своими словами
4. Куда применить
5. Ссылки

## Поля для заполнения

- **Источник и место:** [книга / глава / страницы]
- **Исходная идея:** [НУЖНО ЗАПОЛНИТЬ]
- **Своими словами:** [НУЖНО ЗАПОЛНИТЬ]
- **Термины:** [НУЖНО ЗАПОЛНИТЬ]
- **Куда применить:** тема / билет / проект
- **Связанные файлы:** [ссылки]
- **Уверенность:** высокая / средняя / низкая

## Связь с другими разделами

Результат сохраняется в Knowledge OS (`08`) и при необходимости в карточки темы/билета.
