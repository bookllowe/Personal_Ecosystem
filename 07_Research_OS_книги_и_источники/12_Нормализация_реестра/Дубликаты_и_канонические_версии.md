# Дубликаты и канонические версии

## Назначение

Файл фиксирует группы дубликатов и выбирает, с каким файлом работать дальше.

## Правило выбора канонической версии

Если есть несколько одинаковых или близких файлов, канонической считается версия:

1. с лучшим форматом для обработки;
2. с меньшим размером при сохранении качества;
3. с читаемым текстовым слоем;
4. с понятным названием;
5. без дублирования DJVU и PDF, если PDF уже читается;
6. с актуальной или полной версией.

**Важно:** файлы не удаляются. Фиксируется только рабочая версия.
