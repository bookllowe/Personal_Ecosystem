# Партия 1.1. Экзаменационный минимум

## Назначение

Партия 1 содержит слишком много файлов. Для реальной обработки нужно выделить меньшую подгруппу — Партию 1.1.

## Принцип отбора

В Партию 1.1 входят 15–25 источников, которые первыми нужны для билетов 1–30:

- общая педагогика;
- воспитание;
- методика обучения истории;
- школьное историческое образование;
- ФГОС / ФОП / ФРП / нормативные документы;
- диагностика, оценивание, педагогические измерения;
- предметные умения по истории;
- работа с источниками на уроке истории;
- наглядность и карта;
- учебник истории.

Если среди файлов есть дубликаты, включается только каноническая версия.
