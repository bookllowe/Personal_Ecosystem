# Алгоритм анализа книги для лекции

## 1. Нельзя читать всё подряд

Перед чтением проводится структурная разведка:

- титульный лист;
- сведения об авторе;
- год и место издания;
- вид издания;
- аннотация;
- оглавление;
- введение;
- заключение;
- предметный указатель;
- библиография;
- релевантные главы;
- таблицы, схемы и приложения.

## 2. Типология источников

| Тип источника | Функция в лекции |
|---|---|
| 1. Базовые учебники | Дают устойчивые определения, общую структуру темы и базовую терминологию |
| 2. Специализированные монографии | Углубляют проблему, показывают авторскую концепцию и научную дискуссию |
| 3. Научные статьи | Дают современные подходы, спорные вопросы и актуальные акценты |
| 4. Первоисточники | Обеспечивают доказательность и работу с историческим материалом |
| 5. Нормативные документы | Связывают тему с требованиями образования, стандартами и программами |
| 6. Энциклопедии и справочники | Уточняют понятия, факты, персоналии и устойчивые формулировки |
| 7. Методические пособия | Переводят теорию в практику преподавания истории |
| 8. Статистические и аналитические материалы | Дают данные, динамику и опору для аргументов |

## 3. Что извлекать из книги

- определения;
- ключевые тезисы;
- классификации;
- признаки;
- функции;
- этапы;
- модели;
- причинно-следственные связи;
- аргументы;
- примеры;
- факты;
- даты;
- позиции учёных;
- противоречия;
- выводы;
- таблицы;
- схемы;
- цитаты;
- вопросы для обсуждения.

## 5. Порядок извлечения материала

1. Поиск ключевых терминов.
2. Поиск релевантных глав.
3. Быстрое диагностическое чтение.
4. Глубокое чтение.
5. Извлечение атомарных тезисов.
6. Сопоставление источников.
7. Синтез учебного текста.

## 6. Что нельзя делать

- Брать одну книгу за абсолютную основу.
- Копировать крупные фрагменты.
- Соединять цитаты без объяснения.
- Переносить структуру одной книги в лекцию без проверки.
- Смешивать определения разных авторов.
- Скрывать научные разногласия.
- Использовать устаревшие сведения как современные.
- Ссылаться на источник, который не был просмотрен.
- Придумывать страницы и цитаты.
- Делать вывод, который не следует из доказательств.

## Связь с другими разделами

Результаты анализа идут в карточки книг и глав (`07`), затем в полную лекцию (`04`, `14`) и в ответы по билетам (`06`).
