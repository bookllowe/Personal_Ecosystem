# Правила работы с 281 файлом

## Главное правило

Не читать все книги подряд.

## Нельзя

- отправлять весь список книг в один анализ без структуры;
- конвертировать все книги сразу;
- делать OCR всех файлов сразу;
- создавать ответы по билетам до карты источников;
- использовать книгу в ответе без фиксации страницы или главы;
- повторно отправлять уже обработанные главы.

## Можно

- сначала собрать полный реестр;
- классифицировать по названиям;
- выделить приоритеты;
- обрабатывать партиями;
- конвертировать только нужные главы;
- сохранять промежуточные результаты;
- фиксировать цитаты и страницы;
- создавать тезисы по главам;
- потом собирать ответ по билету из уже сжатых материалов.

## Схема экономии токенов

281 файл
↓
реестр
↓
классификация
↓
10–20 приоритетных книг
↓
оглавления
↓
карты глав
↓
нужные главы
↓
Markdown
↓
тезисы
↓
билет

## Размер порций

1. Реестр — только имена файлов.
2. Оглавление — только структура книги.
3. Глава — 5–15 страниц за раз.
4. Тезисы — одна глава или один раздел.
5. Билет — только релевантные тезисы.
6. Устный ответ — только финально сжатый материал.
