# Маршруты обработки файлов библиотеки

## Главное правило

281 файл нельзя обрабатывать одинаково. Файлы делятся на маршруты.

## Маршрут A. Текстовый PDF

Процесс:
PDF → оглавление → карта глав → выбор глав → Markdown → тезисы → билет.

## Маршрут B. PDF-скан

Процесс:
PDF → OCR → проверка OCR → оглавление → Markdown по главам → тезисы.

## Маршрут C. PDF с таблицами и схемами

Процесс:
PDF → оглавление → ручная проверка таблиц → выбор глав → Markdown + отдельные таблицы.

## Маршрут D. DOC/DOCX

Процесс:
DOC → DOCX → Markdown → оглавление → карта глав → тезисы.

## Маршрут E. DJVU

Процесс:
DJVU → PDF или OCR → проверка текста → Markdown.

## Маршрут F. Изображения

Процесс:
изображение → OCR при необходимости → ручная проверка.

## Маршрут G. Архивы

Процесс:
архив → распаковка вручную → повторная инвентаризация.

## Маршрут H. Неясные файлы

Процесс:
ручная проверка → решение о включении в систему.
