# Матрица поиска по книгам для билетов 1–9

## Назначение

Файл показывает, какие понятия, главы и разделы нужно искать в книгах для подготовки билетов 1–9.

## Главное правило

Книга анализируется только по тем блокам, которые нужны для билета.

## Как использовать эту матрицу

1. Перед открытием книги выбрать билет.
2. Посмотреть, какие понятия и блоки нужны.
3. Найти в оглавлении главы по этим словам.
4. Не читать книгу целиком.
5. Извлечь только нужные главы или фрагменты.
6. Сохранить тезисы и страницы.
7. Потом собрать ответ.
