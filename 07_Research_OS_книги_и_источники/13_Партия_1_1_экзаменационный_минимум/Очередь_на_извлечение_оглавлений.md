# Очередь на извлечение оглавлений

## Назначение

Файл определяет, какие источники Партии 1.1 переходят к Промпту 7.

## Группы очереди

### Группа A. Оглавление уже есть

Файлы, которые не нужно обрабатывать повторно.

- Короткова / Студеникин (схемы и таблицы) — оглавление в `03_Оглавления`
- Селиванова — оглавление в `03_Оглавления`
- Общая педагогика Учеб. пособие.pdf — дубль Загвязинского (не извлекать отдельно)

### Группа B. Готовы к извлечению оглавления

Файлы, где текстовый слой есть и можно извлекать оглавление.

- Загвязинский, Емельянова. Общая педагогика
- Вяземский, Стрелова [2001]
- Глазырина [2009]
- Мезенцев
- Подласый
- Милач [2013]
- Лейбова, Хлытина
- Приказ № 413 (нормативный режим структуры)
- Статья о предметных умениях (№22)
- Статья о наглядности (№23; учесть таблицы/схемы)

### Группа C. Сначала техническая проверка

Файлы крупные, подозрительные или с маршрутом A/B, B?, C.

- Барабанов (битая кодировка)
- Педагогические измерения (смешанная кодировка)
- Голиков (131 МБ; текст есть — только точечно)
- Головеева (51 МБ / 707 стр.; «Содержание» в файле — точечно)
- Студеникин учебник (нет текста → OCR)
- Приказы 653, 732, 569 (сканы → OCR / текстовые копии)

### Группа D. DOC/DOCX

Файлы Word, где надо проверить структуру и извлечь оглавление.

- Лихачёв Б.Т. Курс лекций по педагогике.docx
- Федеральная образовательная программа.docx (нормативный режим)

### Группа E. Нормативные документы

ФГОС, ФОП, ФРП, приказы. Для них нужен отдельный режим извлечения структуры.

- ФОП.docx
- Приказ 413
- Приказы 653, 732, 569 (после появления текста)

## Рекомендуемый порядок в Промпте 7

1. Группа B (быстрый выигрыш для билетов 1–30).
2. Группа D (Лихачёв + ФОП).
3. Группа C точечно: Головеева (есть «Содержание»), затем Голиков.
4. Группа C проблемные: Барабанов, Педагогические измерения.
5. Группа C/E сканы: приказы 653/732/569 и Студеникин — только после OCR/текстовой копии.
6. Группу A не трогать.

## Использование обязательного содержания билетов 1–9

Перед извлечением оглавлений книг по педагогике нужно сверяться с файлом:

`07_Research_OS_книги_и_источники/13_Партия_1_1_экзаменационный_минимум/Матрица_поиска_по_книгам_для_билетов_1_9.md`

Оглавление извлекается не ради общего конспекта, а чтобы найти главы, закрывающие обязательные блоки билетов 1–9.

## Использование обязательного содержания билетов 10–13

Перед извлечением структуры нормативных документов нужно сверяться с файлом:

`07_Research_OS_книги_и_источники/13_Партия_1_1_экзаменационный_минимум/Матрица_поиска_по_документам_для_билетов_10_13.md`

Для нормативных документов извлекается не обычное оглавление, а структура:
- разделы о целях;
- разделы о результатах;
- разделы о содержании истории;
- разделы о тематическом планировании;
- разделы о воспитании;
- разделы о рабочей программе.
