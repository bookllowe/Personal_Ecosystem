# Матрица поиска по книгам для билетов 14–21

## Назначение

Файл показывает, какие понятия, главы и разделы нужно искать в книгах по методике преподавания истории для подготовки билетов 14–21.

## Главное правило

Книга анализируется только по тем блокам, которые нужны для билета. Для этой группы особенно важно искать не общий пересказ методики, а конкретные разделы о содержании исторического образования, знаниях, умениях, представлениях, времени, пространстве и понятиях.

## Согласование с формулировками билетов

Формулировки билетов взяты из `06_Exam_OS_экзамены_и_билеты/Список_билетов.md`:
- 14 — Структура учебного материала;
- 15 — Исторические знания школьников;
- 16 — Формирование и развитие умений учащихся в процессе обучения истории;
- 17 — Проблемы воспитания в процессе обучения истории;
- 18 — Исторические представления;
- 19 — Представления учащихся об историческом времени;
- 20 — Представления учащихся об историческом пространстве;
- 21 — Исторические понятия.

## Как использовать эту матрицу

1. Сначала выбрать билет 14–21.
2. Посмотреть, какие блоки нужно раскрыть.
3. Открывать оглавления методических книг не целиком, а под ключевые слова.
4. Не делать общий конспект книги.
5. Извлекать только нужные главы и страницы.
6. Фиксировать, какой блок билета закрывает каждая глава.
7. Потом создавать карту глав.
8. Потом выбирать главы для Markdown.
