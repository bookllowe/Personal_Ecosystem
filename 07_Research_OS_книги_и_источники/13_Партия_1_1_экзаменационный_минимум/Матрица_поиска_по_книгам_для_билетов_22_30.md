# Матрица поиска по книгам для билетов 22–30

## Назначение

Файл показывает, какие понятия, главы, разделы, методические приёмы, средства, задания и примеры нужно искать в книгах по методике обучения истории для подготовки билетов 22–30.

## Главное правило

Книга анализируется только по тем блокам, которые нужны для билета. Для этой группы особенно важно искать практические методические элементы: классификации, приёмы, средства, алгоритмы работы, примеры уроков и задания.

## Как использовать эту матрицу

1. Сначала выбрать билет 22–30.
2. Посмотреть, какие блоки нужно раскрыть.
3. Открывать оглавления методических книг только под ключевые слова.
4. Не делать общий конспект книги.
5. Извлекать только нужные главы и страницы.
6. Фиксировать, какой блок билета закрывает каждая глава.
7. Потом создавать карту глав.
8. Потом выбирать главы для Markdown.
9. Потом делать тезисы.
10. Потом собирать ответ.
