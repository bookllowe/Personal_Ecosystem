# Реестр книг для подготовки билетов

## Назначение

Этот файл фиксирует все книги, которые используются для подготовки экзаменационных билетов.

## Источник оригиналов

```text
C:\Users\User\Documents\Книги для магистратуры
```

## Главное правило

Сначала книга регистрируется и диагностируется. Только потом выбираются нужные главы. Полная конвертация всей книги допускается только после решения, что книга является базовой.

## Статусы книги

- обнаружена во внешней папке;
- зарегистрирована;
- требует технической диагностики;
- текстовый PDF;
- скан без текстового слоя;
- плохой OCR;
- требуется OCR;
- DOC требует конвертации;
- DOCX готов к конвертации;
- оглавление извлечено;
- главы размечены;
- связана с билетами;
- частично конвертирована в Markdown;
- полностью конвертирована в Markdown;
- тезисы извлечены;
- готова к использованию в билетах.
