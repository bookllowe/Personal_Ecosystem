# Карта системы

## Главная логика

Источник → Извлечение → Структурирование → Лекция → Ответ по билету → Устная версия → Самопроверка → Комиссия → Портфолио

## Учебная ветка

Тема → Категория → Лекция → Мини-проект → Навык → Компетенция

## Экзаменационная ветка

Предмет → Группа билетов → Билет → Литература → Лекция → Ответ → Вопросы комиссии → Повторение

## Исследовательская ветка

Книга → Глава → Тезисы → Понятия → Сравнение источников → Синтез → Лекция

## Жизненная ветка

Сфера жизни → Задача → Нагрузка → Малый шаг → Прогресс → Стабилизация

## Проектная ветка

Идея → Категория → Архитектура → Карточка проекта → Мини-проект → Прототип → Web/HTML

## Портфолио

Документ → Тип достижения → Критерий комиссии → Где использовать в ответе → Что показать комиссии
