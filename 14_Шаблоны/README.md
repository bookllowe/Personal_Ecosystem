# 14_Шаблоны

## Назначение раздела

Готовые шаблоны README, лекций, ответов, анализа тем и проектов.

## Что хранится в разделе

- Шаблоны документов для копирования в рабочие разделы

## Основные файлы

- `README.md`
- `Шаблон_README.md`
- `Шаблон_лекции.md`
- `Шаблон_полной_академической_лекции.md`
- `Шаблон_мини_проекта.md`
- `Шаблон_таблицы_источников.md`
- `Шаблон_карты_категории.md`
- `Шаблон_устного_ответа.md`
- `Шаблон_ответа_по_билету.md`
- `Шаблон_анализа_темы.md`

## Связь с другими разделами

`Шаблон_лекции.md` — опорный; полная лекция — только через `Шаблон_полной_академической_лекции.md`.

## Как экономить токены

- Хранить длинные данные в markdown-файлах.
- Не отправлять в ИИ всю книгу целиком.
- Анализировать документы по главам.
- Использовать краткие карточки.
- Повторно использовать шаблоны.
- Хранить промежуточные результаты.
