# 02_Словарь_архитектора

## Назначение раздела

Единые определения терминов архитектуры, знаний, AI, визуализации, педагогики и методики.

## Что хранится в разделе

- Общий словарь
- Тематические списки терминов

## Основные файлы

- `README.md`
- `Словарь_терминов.md`
- `Термины_архитектуры.md`
- `Термины_баз_знаний.md`
- `Термины_AI.md`
- `Термины_визуализации.md`
- `Термины_педагогики_и_методики.md`

## Связь с другими разделами

Питается Атласом, лекциями и билетами; используется в ответах комиссии.

## Как экономить токены

- Хранить длинные данные в markdown-файлах.
- Не отправлять в ИИ всю книгу целиком.
- Анализировать документы по главам.
- Использовать краткие карточки.
- Повторно использовать шаблоны.
- Хранить промежуточные результаты.
