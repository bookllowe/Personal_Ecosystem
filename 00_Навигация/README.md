# 00_Навигация

## Назначение раздела

Быстрый вход в систему: карта, старт, указатели и ежедневный маршрут.

## Что хранится в разделе

- Главная карта
- Быстрый старт
- Указатель размещения
- Ежедневный маршрут

## Основные файлы

- `README.md`
- `Главная_карта.md`
- `Быстрый_старт.md`
- `Где_что_лежит.md`
- `Ежедневный_маршрут.md`

## Связь с другими разделами

Связана со всеми разделами. Подробная логика — в корневой `КАРТА_СИСТЕМЫ.md`.

## Как экономить токены

- Хранить длинные данные в markdown-файлах.
- Не отправлять в ИИ всю книгу целиком.
- Анализировать документы по главам.
- Использовать краткие карточки.
- Повторно использовать шаблоны.
- Хранить промежуточные результаты.
