# 04_Программа_обучения

## Назначение раздела

План курса, этапы, стандарты урока, лекции, мини-проекта и место практики.

## Что хранится в разделе

- Программа и этапы
- Стандарты лекции/урока/мини-проекта
- Чеклисты

## Основные файлы

- `README.md`
- `Программа_курса.md`
- `Этапы_обучения.md`
- `Чеклист_урока.md`
- `Формат_лекции.md`
- `Стандарт_академической_лекции.md`
- `Стандарт_урока.md`
- `Стандарт_мини_проекта.md`
- `Место_практики_в_обучении.md`

## Связь с другими разделами

Лекции длинные (≥15 минут чтения). Практика — только после теории и проектирования.

## Как экономить токены

- Хранить длинные данные в markdown-файлах.
- Не отправлять в ИИ всю книгу целиком.
- Анализировать документы по главам.
- Использовать краткие карточки.
- Повторно использовать шаблоны.
- Хранить промежуточные результаты.
