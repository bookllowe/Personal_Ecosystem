# Чеклист урока

## Назначение

Контрольный список: урок подготовлен, проведён и закреплён.

## Структура

1. До урока
2. Во время урока
3. После урока

## Поля для заполнения

**Урок:** [НУЖНО ЗАПОЛНИТЬ]  
**Дата:** [НУЖНО ЗАПОЛНИТЬ]

### До урока
- [ ] Тема зафиксирована в Атласе
- [ ] Цель урока ясна
- [ ] Источники выбраны
- [ ] Карточка урока создана
- [ ] Материалы подготовлены

### Во время урока
- [ ] Ключевые понятия названы
- [ ] Есть примеры
- [ ] Есть схема / визуализация
- [ ] Тема раскрыта развёрнуто, а не только тезисами
- [ ] Есть вопросы для проверки и возможные вопросы комиссии

### После урока
- [ ] Резюме сохранено
- [ ] Термины добавлены в Словарь
- [ ] Карточка обновлена
- [ ] Следующий шаг записан

## Связь с другими разделами

Используется вместе с `Формат_лекции.md`, `Карточка_урока.md` (`05`) и Knowledge OS (`08`).
