# Архитектура базы знаний

## Назначение

Как устроена база знаний: слои, типы заметок, связи.

## Структура

1. Слой источников
2. Слой извлечений
3. Слой понятий
4. Слой синтезов / лекций
5. Слой применения (билеты, проекты)

- **Основные типы заметок:** [НУЖНО ЗАПОЛНИТЬ]
- **Правило связей:** [НУЖНО ЗАПОЛНИТЬ]
