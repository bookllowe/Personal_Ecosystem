# 05_Карточки_обучения

## Назначение раздела

Краткие карточки тем, категорий, инструментов, архитектуры, проектов, исследований, уроков и лекций.

## Что хранится в разделе

- Навигационные карточки без длинных текстов

## Основные файлы

- `README.md`
- `Карточка_темы.md`
- `Карточка_категории.md`
- `Карточка_инструмента.md`
- `Карточка_архитектуры.md`
- `Карточка_проекта.md`
- `Карточка_исследования.md`
- `Карточка_урока.md`
- `Карточка_лекции.md`

## Связь с другими разделами

Карточки краткие; полные лекции и ответы хранятся в `04`, `06`, `08`.

## Как экономить токены

- Хранить длинные данные в markdown-файлах.
- Не отправлять в ИИ всю книгу целиком.
- Анализировать документы по главам.
- Использовать краткие карточки.
- Повторно использовать шаблоны.
- Хранить промежуточные результаты.
